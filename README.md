# 🚴 Bike Purchase Analysis (Excel)

## 📊 Overview
This project analyzes a bike sales dataset to understand customer purchasing behavior and identify key factors influencing bike purchases. The analysis was conducted using Microsoft Excel and presented through an interactive dashboard.

---

## 📁 Dataset
The dataset includes customer demographic and behavioral information such as:
- Income  
- Age group  
- Occupation  
- Commute distance  

---

## 🔄 Process
- Data cleaning and preprocessing  
- Data exploration using PivotTables  
- Creation of visualizations (charts)  
- Development of an interactive dashboard  

---

## 🔍 Key Insights
- Higher income levels are associated with increased bike purchases  
- Certain age groups show stronger purchasing trends  
- Commute distance impacts buying decisions  
- Customer demographics influence purchasing behavior  

---

## 🛠️ Tools Used
- Microsoft Excel  
- PivotTables  
- Charts & Data Visualization
